What is blockem?

Blockem is an open source clone of the board strategy game blokus in its duo version. It is available for Linux and Windows and written in C++/GTKmm (the C++ wrapper of GTK+) and hosted in sourceforge


Rules:

Basically you've got 21 pieces to put down on the board (which are a set of all the possible free polyominoes from 1 to 5 squares), while preventing your opponent from doing the same. There are 3 basic rules to put down pieces on the board:

  • The first piece played of each color is placed 5 squares far from one of the board's four corners. The opponent starts 5 squares far from the opposite corner.
  • Each new piece played must be placed so that it touches at least one piece of the same color, with only corner-to-corner contact allowed. (Edges cannot touch).
  • When a player cannot place a piece, he or she passes, and play continues as normal. The game ends when no one can place a piece

HOWTO compile blockem sources:

Blockem sources has been compiled in both Linux and Windows, however it should compile in every platform where GTKmm has been ported.

Linux:

Compiling blockem in a linux environment is "almost" straight away, you'll need to have the g++ compiler and the following dev libraries:
  • gtkmm-2.4
  • glademm-2.4
In a debian based linux (i.e. ubuntu) run 'apt-get install g++ libgtkmm-2.4-dev libglademm-2.4-dev'.

Now your system is ready to compile the package.

  1. Download the latest tarball
  2. uncompress it (tar -zxf blockem-x.y.z.tar.gz)
  3. cd into it (cd blockem-x.y.z)
  4. run './configure ∓& make'
  5. run 'make install' as root and blockem will be installed in your default binary directory (tipically /usr/bin)

Windows:

There is a ready-to-run precompiled binary in the release list of files. If you still want to compile the sources in this platform have a look at this HOWTO I wrote when I compiled it for the 1st time.
